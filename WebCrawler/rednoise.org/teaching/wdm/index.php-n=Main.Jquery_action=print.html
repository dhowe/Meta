An Introduction to jQuery

jQuery is a hugely popular Javascript library for manipulating web page with ease. You can find anything you want in a page (using the DOM, or Document-Object-Model) and then use jQuery to show, hide, and style elements, react to user actions, and add effects. The first thing to do with jQuery is to SELECT the item that you want to manipulate.

TO SELECTUSE
All elements$("*")
Element type (example: <img>)$("img")
ID (example: id="myID")$("#myID")
Class (example: class="myClass")$(".myClass")
Order (example: the first or last <img> element)$("img:first") $("img:last")
Attribute (example: the 'href' attribute of <a>)$("a[href]")
Parent (example: the parent of <div>)$("div:parent")
Child (example: the first or last child of <div>)$("div:first-child") $("div:last-child")

For more selectors, see this page.



Inserting Content In/Around Elements with jQuery

jQuery lets you access and modify the data on a web page. You can insert (or replace) content wherever you want, in relation to an element, for example, a <div>:

TO PLACE USE
Inside the <div>, replacing what's there (text)
$("div").text("<em>theText</em>");
Inside the <div>, replacing what's there (html)
$("div").html("<em>theHtml</em>");
In front of and outside the <div> (before)
$("div").before("Before<br/>");
In back of and outside the <div> (after)
$("div").after("After<br />");
Inside the <div> but before existing content (prepend)
$("div").prepend("New Content!");
Inside the <div> but after existing content (append)
$("div").append("New Content!");



To use jQuery, you need to include it in your HTML page as follows (it can also be installed locally). You then add your Javascript code inside the 'ready' function (generally-speaking).

<html>
<head>
<title>Test Page</title>
<script src="http://code.jquery.com/jquery-latest.min.js"></script>
<script type="text/javascript">
$(document).ready(function(){

// Your code goes here.

});
</script>
</head>

<body>
<div>I am a div.</div>
<br/>
<input value="Go" type="submit">
</body>

</html>
