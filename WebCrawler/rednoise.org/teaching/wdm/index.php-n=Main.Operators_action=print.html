Operators

Assignment Operator: '='

Some Examples (test in p5.js if unclear): 

  let x = 7;
  let y = 3 + 4;
  let f = 7.1; 
  let s = "seven";

Mathematical Operators: +, -, /, * (addition, subtraction, division and multiplication)

Some Examples (test in p5.js if unclear): 
        let aNumber = 0;

        aNumber = aNumber + 1;
        println("aNumber = " + aNumber);

        aNumber = aNumber + aNumber;
        println("aNumber = " + aNumber);

        aNumber = aNumber/2;
        println("aNumber = " + aNumber);

        aNumber = 7 * 8;
        println("aNumber = " + aNumber);

        // "modulus" returns the 'remainder' after a division
        aNumber = aNumber % 5; 
        println("aNumber = " + aNumber);

        aNumber++;  // Short-cut for adding one, called "increment"
        println("aNumber = " + aNumber);

        aNumber--; // Short-cut for subtracting one, called "decrement"
        println("aNumber = " + aNumber);

Relational Operators: 

  >= (greater than or equal to)
  <= (less than or equal to)
  == (equality)
  != (inequality)

Logical Operators: Sometime called "boolean" or "logical" operators: 

  || (logical OR)
  && (logical AND)
  !  (logical NOT)
