Arrays

An Array is simply a list of variables of a specific type (like those mentioned earlier: float, int, String, etc.) We can access each element of the list via its 'index', an integer value that uniquely designates its position in the list. For example, 'myStrings[4]' refers to index 4 of the myStrings array (the 5th element since we know that arrays start at 0). Arrays are especially useful when we have a large number of some type of variable and want to do the same thing to each (most often using a 'for' loop; see loops).

Declaring Arrays

An array-type variable is declared with a type (float, int, String, etc.) followed by empty square brackets []. 

 int[]    arrayOfInts;
 float[]  arrayOfFloats;
 String[] arrayOfStrings;
 RiText[]  arrayOfRiTexts;

We have to use the "new" operator to create an instance of an array (this is the moment where the memory required to store the information inside the array is allocated.) Also, when creating the array, we have to define how many elements we want the array to hold. We use the 'new' operator, followed by the data type, followed by a bracketed integer expression (indicating the total # of elements). 

 arrayOfInts = new int[42];
 arrayOfInts = new int[x + 5];

If we know what the values we want the array to start with, we can also write include these as list enclosed in curly brackets and separated by commas (here the 'new' is implicit). 

 int[] arrayOfInts    = {1, 5, 8, 9, 4, 5};
 String[] arrayOfStrs = {"hello", "bye", "HI"};

Using Arrays

We refer to any element of the array by the name of the array, followed by an index number enclosed in brackets. In Java/Processing, we consider the first element of the array as having index #0. 

 int[] stuff = new int[5];
 stuff[0] = 5;
 stuff[1] = 5;
 stuff[2] = 5;
 stuff[3] = 5;
 stuff[4] = 5;

Iteration

An array is a powerful tool given its ability to store multiple pieces of information all together in sequence. We discover just how powerful by performing identical operations on each element of the array. Imagine we wanted to add 3 to each element in our 'stuff' array. Certainly we could do this: 

 stuff[0] = stuff[0] + 3;
 stuff[1] = stuff[1] + 3;
 stuff[2] = stuff[2] + 3;
 stuff[3] = stuff[3] + 3;
 stuff[4] = stuff[4] + 3;

A little tedious perhaps. But what if our array contained 1000 elements? Would we really want to write out 1000 lines of code for this simple operation? How can we "iterate" through each element of the array?

Well, we can use a loop. Here we loop through the 'stuff' array (of whatever length*), adding 3 to each, using a 'while' loop. 

 int index = 0;
 while (index < stuff.length) {
   stuff[index] += 3;
   index++;
 }

NOTE*: the length of any array is stored in its 'length' variable: e.g., array.length, or in the case above: 'stuff.length'; this is how we tell the loop to stop before going past the end of the array.
