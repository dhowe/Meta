   Assignment #3 (Due Mon. Feb 22 )      

1. Reading
  • Read the P. Galanter article: 'What is Generative Art? Complexity Theory as a Context for Art Theory', which should directly inform your group project described below).
Keep in mind that although you have 2 weeks for the these projects, there will be additional reading/writing/viewing assignments given for next week.
  • Create (with your partner) a work of generative literary art using one of the types of systems mentioned in the Galanter article. The constraints for this assignment are as follows:
    • Text must be the primary visual element. Other visual media (images, video, etc.) should be used only as a means of framing or contextualizing the textual.
    • The text should not only be decorative, but should make linguistic (and hopefully literary sense). To be more specific, it is NOT Ok, for example, to take a flocking system and simply substitute random letters or words for the points.
    • You should however feel free to use an existing generative simulation (there are MANY on the Processing website) as a starting point from which to develop your literary system.
Again, make sure there is conceptual link between the content (the text), the form (the display and/orinteraction) and the generative mechanism you choose.
  • Publish a short description (at least one sentence), a 'thumbnail' (250x100), and a link to your sketch under your name on this page, no later than Monday at 4pm.
  • Misc: make sure you have your copy of Hartman's 'Virtual Muse'!
