Some Html you can use for the examples that follow...

<html>
<head>
<title>My Test Page</title>
<script src="libs/jquery-1.4.min.js"></script>
<script>
$(document).ready(function(){

//Your code goes here.

});
</script>
</head>
<body>
<div>I am a div.</div>
<br />
<input value="Go" type="submit">
</body>
</html>
[$[Get Code]]

Some examples showing how to insert content in/around a <div>. Grab the code from below (under USE) and paste it into the HTML above.


TO PLACE USE

Inside the <div>, replacing what is there (html)

$(":submit").click(function () { 

  $("div").html("<em>Some Html</em>");

});

Inside the <div>, replacing what is there (text)

$(":submit").click(function () { 

  $("div").text("<em>Some Text</em>");

});

In front of and outside the <div> (before)

$(":submit").click(function () { 

  $("div").before("Before<br/>");

});

In back of and outside the <div> (after)

$(":submit").click(function () { 

  $("div").after("After<br />");

});

Inside the <div> but before existing content (prepend)

$(":submit").click(function () { 

  $("div").prepend("New Content!");

});

Inside the <div> but after existing content (append)

$(":submit").click(function () { 

  $("div").append("New Content!");

});
