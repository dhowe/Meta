Assigment #8 (Due 4/17)

Reading

-- Read/View Tom Phillips' piece: A Humument.

-- Read the four short essays on 'A Humument' HERE. Be prepared to discuss specific points raised by each in class.

-- Browse Aya Karpinska's website and read the short interview with her here. On Tuesday, she'll be showing and talking about some of her new work.

Writing

-- If you have not done so, pls. send me your write-up of 3 ideas for a final project. These can be variations on a single theme or 3 completely separate projects from which you will choose. Include sketches, storyboards, images, and sounds when possible. If you haven't already, be prepared to read/show your proposals in class and to give feedback on those of others.

-- N-Grams: Compose a 'literary' text from the output of your N-Gram sketch (see below). This is a collection of lines/phrases output from the sketch which you've selected and/or edited into some cohesive unit (a poem, short story, monologue, dramatic dialog, etc.) This piece should stand on its own apart from the sketch and/or any explanation of process.

Programming

-- Complete the N-Gram assignment as discussed in class. This involves 2 components:

1) Create a processing sketch using the PMarkov object (see the example). Do not simply change the input texts, but instead modify the sketch (line-length, lines-per-page, colors, motion, sound, etc) so that the visual presentation reflects the 'content' of the piece in some way.
2) Compose a 'literary' text from the generated output of the sketch (see above).

-- Be sure to download and install the current version of pText (013) if you haven't already.
