    Assignment #1 (due Jan 20, 12pm)

  • Create a website (or blog) for the course and submit the LINK on Canvas (if the page has a password, include it). This page will eventually hold links to ALL OF YOUR WORK for the course, so design it with this in mind. The top of your first page MUST include:
    • your name (and nickname if you have one)
    • your student ID #
    • your CityU email address
    • links to each assignment
The first link should be a response (between 1-3 paragraphs) to this reading?. It may be a disagreement with the text, a question you want answered, a provocation, or even a mini-artwork addressing the article's themes. Do not simply summarize the contents of the article.
