   Assignment #1 (Due Mon. Feb 8)

1. TOOLS AND SETUP
  1. Download & install the Processing environment here
  2. Verify you can run the examples in your Sketchbook:
    Menu: File->Examples
  3. Download & install the RiTa core library (v96)

    Download and unzip RiTa.zip from here to your desktop. Then create a 'libraries' directory (if it doesn't already exist) inside your Processing sketchbook and place the 'rita' folder inside. (follow this same procedure to install other Processing libraries)
  4. Verify you can run the RiTa examples in your Sketchbook:
    Menu: File->Sketchbook->Libraries->RiTa->Examples

    Go through each and try to understand what is going on.
    Keep track of any questions you can't resolve on your own.
  5. Visit and bookmark the following reference pages:
    • the (extended) Processing reference:
      http://processing.org/reference/index_ext.html
    • the RiTa reference and javadocs:
      http://rednoise.org/rita/documentation/docs.htm
      http://rednoise.org/rita/javadocs/index.html
    • the Java API reference:
      http://java.sun.com/j2se/1.5.0/docs/api/


2. PROGRAMMING REVIEW/CONCEPTS
Read through the ProgrammingConcepts pages on the topics below and make sure you understand the concepts (much of this may be review; if so, continue down the page.) Come to class with at least 2 (written) questions that came up for you during the reading (and/or installation and running of RiTa/Processing examples)
  • Operators, Variables, Arrays
  • Conditionals, Program Flow, Loops,
  • Functions, Classes and Objects


3. DEVELOPING A CRITICAL VOCABULARY
Locate a Processing (or Java) project that uses text (the Exhibition section is a good place to start). Write up a short description (<1pg) and be prepared to present the project ot the class. In addition to describing what you like about it and how it works, be sure to address specific conceptual, technical, and aesthetic aspects of the work. (If possible, pick a project with links to its source code)


4. READING
  • Read the short chapter by Paul Graham entitled 'Hacking and Painting' (pdf)

    (Optional -- A nice intro to the philosophy of making art in Processing:
       'Creative Coding' by Ira Greenberg (pdf) from
       Processing: Creative Coding and Computational Art")
  • Find a copy of the required text for the course:
                Virtual Muse by Charles Hartman.
  • Start thinking about project ideas...



Instructor: Dr. Daniel C. Howe
