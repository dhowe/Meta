Audio Libraries:

Minim

http://code.compartmental.net/tools/minim/

- JavaSound API

- good descriptions, can get source code

- variety of functions: audio input, sample playing, synthesis, effects, FFT, audio recording

- also: beat detection

- direct access to sample data

- can write your own effects

- can play audio in mono and stereo

Ess

http://www.tree-axis.com/Ess/

- JavaSound API

- audio input, playback, synthesis, effects, FFT, audio recording

- more effects than Minim

- more complex FFT functions

- can also do streaming of audio

- slightly confusing class descriptions

- ALSO: info on how to sign an applet (needed for using microphone input)

- audio is mono

Sonia

http://sonia.pitaru.com/

- uses JSyn plugin

- audio input (only for FFT analysis), audio playback, synthesis, FFT, audio recording

- audio can be mono or stereo

RiSample

http://www.rednoise.org/rita/documentation/index.htm

RiSpeech
