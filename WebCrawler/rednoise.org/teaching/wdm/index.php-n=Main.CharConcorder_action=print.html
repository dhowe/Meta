Main: CharConcorder


An exercise on data structures and probability

  • Write a program that loads an input text (from the data folder) and prints out a table showing the # of times each character is found in the text. Hint: use an IntDict to store your data (here are some examples).
  • Use the program below as a template. You may add additional functions and code, but do NOT change any of the signatures of the functions provided. Also, do NOT assume a fixed number of (unique) characters in the file.
  • For extra credit, create a visualization of the data for the screen. This can be visual/aural/textual/abstract/interactive, etc. but should makes clear the relative occurrences of the various characters in a more effective way than a simple printout.
  • Send your sketch to me via email (If it is larger than 1MB, post and send a link instead).
